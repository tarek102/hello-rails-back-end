## Getting Started

### Clone this repository

```bash
$ git clone https://github.com/tarek102/hello-rails-back-end.git
$ cd hello-rails-back-end
```

### Run project

```bash
$ rails server

```

### Open page in browser

```bash
$ http://localhost:3000/
```
